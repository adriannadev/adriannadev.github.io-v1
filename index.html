My Skills

Web Development

My journey with HTML, CSS and JavaScript started when I was probably 10 years old and it soon turned into passion. Now I have experience using multiple technologies, both front and back-end including Bootstrap, Node.js with Express.js, Passport as well as using APIs such as Google OAuth and others.
Moreover, I also have experience working in Python (Django) and PHP. I used different databases including MYSQL, SQLite and MongoDB. I also worked with React.js and Angular.

Java

During my time at university we mostly concentrated on Object Oriented Programming in Java. It helped me gain in-depth knwoledge about the language and understand that Java has endless possibilities.
I used this language for projects ranging from image processing applications or simple games to making more complex projects such as an Android application for cyclers that I made with a team of students in Singapore using Google Maps API.

Design

Before becoming a developer I was interested in graphic design and became more passionate about interaction design. I worked on several projects at university using Processing and Arduino. Over the years, I became comfortable using Adobe Creative Suite including Photoshop, InDesign, Illustrator, Lightroom and Premiere Pro.
I think that good design is foundation for anything, as it improves the user experience. Therefore, I find it very rewarding to turn designs into functional products.

A developer based in the UK

In 2018, I graduated with First Class in BSc Computer Science and Multimedia from Queen Mary, University in London. Having studied design at Goldsmiths before, I wanted to combine my creativity with enthusiasm for coding and develop my skills. Currently, I mostly concentrate on front-end development.
